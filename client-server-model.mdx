title: "Client Server Model"
date: "2022-01-16"
tags: ["network", "model"]
---

Client-Server Model (atau juga disebut Request-Response Model) merupakan network architecture yang mendeskripsikan bagaiamana suatu server berinteraksi (guna membagikan resources nya) ke client (Response), dan bagaimana client meminta resource ke server (Request).

Componen dari client-server model adalah Client, server, dan network device.
